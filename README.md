<a name="tr"></a>
## 🇹🇷 Türkçe (Turkish)
### Windows Codec Uyumluluk Çözümü
Bu uygulama, indirilen videoların orijinal Windows Medya Oynatıcı, Filmler & TV ve Akıllı TV'lerde sorunsuz çalışması için otomatik olarak format dönüştürme yapar.

* **🎬 Tam Uyumluluk:** Videoları H.264 (Video) ve AAC (Ses) formatına çevirir.
* **📁 Akıllı Klasör:** Videolar `video_downloader`, sesler `audio_downloader` klasörüne gider.
* **⚡ Hız Takibi:** MiB/s cinsinden hız ve kalan süre (ETA) göstergesi.
* **🛠 Kurulum:** Ana dizine `ffmpeg` klasörünü koyun ve `pip install customtkinter yt-dlp` yazın.

<a name="en"></a>
## 🇺🇸 English
### Windows Codec Compatibility Solution
This app automatically converts downloaded videos to H.264/AAC for 100% compatibility with Windows Media Player, Movies & TV, and Smart TVs.

* **🎬 Full Compatibility:** Fixes "codec not supported" errors by remuxing to MP4.
* **📁 Smart Folders:** Auto-sorts files into `video_downloader` and `audio_downloader`.
* **⚡ Live Stats:** Real-time download speed and ETA tracking.
* **🛠 Setup:** Place `ffmpeg` folder in the root and run `pip install customtkinter yt-dlp`.
